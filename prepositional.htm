The prepositional case is used to indicate location.

Add е to a hard consonant Replace а with е Replace о with е Replace ь (feminine) with и
магазин → магазине улица → удице место → месте церковь → церкови
Replace й with е Replace я with е No change to е Replace ь (masculine) with е
музей → музее семья → семье счастье → счастье камень → камне
Replace ия with ии Replace ие with ии
станция → станции здание → здании

The plural form also changes in the prepositional case:

hard consonant, а, о → ах ь, й, я, е → ях
магазин → магазинах церковь → церковях музей → музеях семья → семьях
улица → удицах счастье → счастьях камень → камнях станция → станциях
место → местах здание → зданиях

ЕГО

When directly preceded by e and directly followed by o, г sounds like в (a v sound instead of g). So, его is pronounced yevo. This is mostly present in the genitive case, as well as the accusative case which uses some genitive pronouns, and the masculine possessive pronoun его which is used in all cases.

Uses

There are a few prepositions which indicate that the corresponding noun will be in the genitive case:

без without у (indicates possession) о(т)from из from для for
кофе без молока у женщины есть кольно далеко от музея она из России это для моего пареня

The genitive case is also used with numbers and amounts:

1 nominative singular 2, 3, 4 genitive singular 5+ genitive plural 0 genitive plural
одна корова два, три, четыре коровы пять или больше коров нуль коров

When dealing with numbers bigger than 20, these rules apply to the last number:

сорок одна 41 корова сто девяносто три 193 коровы двенадцать 12 коров шестдесять 60 коров

The genitive case is also used with more general amounts like много a lot, немного a little, etc.
