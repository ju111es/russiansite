The genitive case is used to indicate possession, negation, origin, and amount- it answers questions like, “whose?”, “what is absent?”, and “of what?”. The genitive case could be considered the most complicated case in Russian, with myriad uses and no real pattern to the ending changes. (So it only gets easier from here!)

The plural form also changes in the gen. case:

Add ов to a hard consonant Remove а Remove о Replace ь (masculine) with ей
волк → волковwolf nom. → wolves gen. женщина → женщинwoman nom. → women gen. яблоко → яблокapple nom. → apples gen. словарь → словарейdictionary nom. → dictionaries gen.
Replace й with ев Replace [vowel] я with й Add й to е Replace ь (feminine) with ей
злодей → злодеевvillain nom. → villains gen. змея → змейsnake nom. → snakes gen. море → морейsea nom. → seas gen. дверь → дверейdoor nom. → doors gen.
Replace ж, ч, ш, щ with ей Replace [consonant] я with ь Replace ие with ий
муж → мужейhusband nom. → husbands gen. кухня → кухонь*kitchen nom. → kitchens gen. помишение → помишенийroom nom. → rooms gen.

*For nouns ending in a, o, or я whose stems end in multiple consonants, an o or e is usually inserted between the consonants in the genitive plural. Кошка → кошек cat nom. → cats gen. and окно → окон window nom. → windows gen. are other examples.

The genitive case is also used with numbers and amounts:

When dealing with numbers bigger than 20, these rules apply to the last number:

ЕГО/ОГО

Many pronouns use the ending его or ого in the genitive case. In these words, г sounds like в (a v sound instead of g). So, его is pronounced yevo; ого, ovo. This is mostly present in the genitive case, as well as the accusative case which uses some genitive pronouns, and the masculine possessive pronoun его which is used in all cases.
